; PrefixMask-style chat review filter
; One comma-separated key=value list per line.
; Supported keys:
; caption       - match window caption mask
; filename      - match full executable path mask
; AppUserModelID - match app user model id (useful for Edge PWA profiles)
; CmdParams     - match command line params
; excludeCaption - skip entries whose caption matches
; excludeFilename - skip entries whose executable path matches
; excludeAppUserModelID - skip entries whose app user model id matches
; excludeCmdParams - skip entries whose command line params match
;
; Behavior:
; - This file is the only source of chat monitor app selection.
; - If this file is empty (or missing), no app is reviewed.
; - If at least one rule exists, an app is reviewed only when it matches at least one rule.
; - Within a single rule line, all populated include keys must match (logical AND / conjunctive).
; - Exclude masks are checked after includes; a match on any exclude mask prevents review.
; - Unread detection is automatic and checks caption for a numeric counter pattern: (\d+)
;
; Examples:

caption=*teams*
filename=*msedge.exe,CmdParams=*Teams*
AppUserModelID=*teams*
excludeCaption=*discord*
excludeCaption=*facebook*
